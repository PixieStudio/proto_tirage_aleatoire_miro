Tirage Aléatoire

Prototypage de tirage aléatoire avant intégration dans un vrai plugin.
Version 0.1 - Fonctions Création de decks multiples et Tirer au hasard un widget.

Le bouton n'est visible que par le propiétaire du board.

Créer les éléments sur le board
  • Importez votre deck de cartes sur le board, ou créez des widgets pour un tirage aléatoire.
    Exemple : 6 Stickers contenant les résultats d'une table aléatoire à 6 entrées.
  • Cliquez sur pour accéder au sous-menu.
    Le bouton vous permet de créer de nouveaux decks, ou de configurer ceux sur le board.
  • Entrez un nom pour votre deck, puis cliquez sur "Créer"
    Une Frame, un bouton pioche, et une zone de tirage vont être créés sur le board.
  • Vous pouvez positionner les nouveaux éléments sur votre board comme bon vous semble.
    Exemple : La frame dans votre espace MJ pour ne pas gêner le chargement, le bouton pioche et la forme de résultat sur votre espace de jeu commun.
    Astuce : Placez un dos de carte devant la zone de résultats pour un effet plus immersif !
Configurer votre deck ou votre table aléatoire
  • Déplacez les images de votre deck ou vos widgets dans la Frame.
  • Cliquez sur le titre de la Frame ou sur la forme accueillant les résultats.
    Les 3 boutons et ne sont actifs qu'en sélectionnant la Frame ou le zone de résultat.
  • Cliquez sur , pour initialiser le base de données du deck.
    Attention : Si vous recliquez sur ce bouton, la base de données sera réinitialisée, l'ancienne pioche ne sera plus en mémoire.
Activer le bouton de pioche
  • Une fois que vous êtes prêt, cliquez sur pour Activer le bouton 'Tirage aléatoire'.
    Info : Cliquer de nouveau sur l'éclair désactivera le bouton de Tirage aléatoire. Cela vous permet de tirer une carte ou un widget aléatoirement, en cliquant sur le bouton du board directement !
  • Si votre deck est vide, ou que vous souhaitez remélanger votre deck, cliquez sur pour Reconstruire le deck initial.
Dans cette version des widgets groupés sont considérés comme un seul widget.
Attention : N'utilisez pas la fonction "Grid" de la Frame de votre deck. Cela désactive la pioche. Si vous le modifiez par erreur, vous pouvez cliquer sur "Freeform" pour résoudre le problème.
